# 技师体系设计

## 核心原则

**只看手艺，不碰擦边。** 平台不参与任何擦边引导、暗示和服务。

## 技师分类

### 1. 店铺技师
- 必须由店铺单位注册，归属于具体店铺
- 店铺负责管理、外派、借调
- 收益结算到店铺账户，由店铺发放

### 2. 散技师（独立技师）
- 个人注册，不归属任何店铺
- 可接上门服务订单
- 可到其他店铺体验、了解实际操作
- 标准、收益要求相对较低

## 认证体系

### 培训认证门店
与链接良好的店铺深度合作，成为官方培训和检验门店。

**认证技师优势：**
- 优先级更高（排序、推荐）
- 收益分成更高
- 可接更多类型订单
- 平台背书，商家更信任

**非认证技师：**
- 可正常接单，但标准和收益较低
- 作为技师池的补充
- 可通过积累评价逐步提升

## 技师池

所有技师（店铺技师 + 散技师）形成一个大的技师资源池。

**店铺可按需筛选：**
- 项目类型匹配
- 技能等级
- 评分区间
- 认证状态
- 距离/可用时间

## 多维度评分体系

### 评分来源
| 评分方 | 场景 | 权重 |
|--------|------|------|
| 顾客 | 服务完成后 | 高 |
| 店铺主 | 借调技师实际操作后 | 中 |
| 培训门店 | 认证考核 | 高 |

### 评分维度
- 手法专业度
- 服务态度
- 准时守约
- 沟通能力
- 卫生习惯

## 上门服务保障

### 技师心理障碍
上门服务对技师来说存在心理门槛，需要从流程上解决：

**安全保障：**
- 订单全程可追踪
- 紧急联系机制
- 服务前/中/后状态上报
- 异常情况快速响应

**标准化流程：**
- 统一着装要求
- 标准服务话术
- 明确服务边界
- 拒绝不当要求的标准流程

**前期策略：**
- 只提供标准化服务项目
- 逐步开放更多服务类型
- 积累安全案例和信任

## 招聘市场功能

平台同时具备"按摩技师领域的58同城/Boss直聘"属性：

### 店铺招工
- 从技师池筛选合适人选
- 查看历史评价、服务数据
- 发布长期/短期招聘需求
- 先借调体验，再决定是否长期合作

### 技师找活
- 浏览店铺招聘信息
- 申请短期借调或长期入职
- 展示个人履历和评价
- 主动联系心仪店铺

### 数据支撑
- 服务次数、好评率
- 擅长项目、技能认证
- 历史合作店铺
- 顾客复购率

## 技师成长路径

```
散技师（新人）
    ↓ 积累评价
散技师（有记录）
    ↓ 通过培训门店认证
认证技师
    ↓ 被店铺长期聘用
店铺技师
    ↓ 表现优异
店铺核心技师 / 培训导师
```

## 收益分配原则

| 技师类型 | 上门服务 | 到店服务 | 外派借调 |
|----------|----------|----------|----------|
| 认证技师 | 高分成 | - | 高分成 |
| 非认证散技师 | 低分成 | - | 低分成 |
| 店铺技师 | 店铺决定 | 店铺决定 | 店铺决定 |

店铺技师的收益由店铺内部规则决定，平台不干预。
