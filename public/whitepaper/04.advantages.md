# 核心优势

## 差异化竞争力

轻养到家相比现有平台，具有以下核心优势：

## 1. 技师借调机制

### 痛点解决

传统模式下，商家面临：
- 旺季缺人，订单流失
- 淡季养人，成本高企
- 临时招人，质量难保

### 我们的方案


```mindmap
# 技师借调机制
## 店铺A（缺人）
- 发起借调需求
- 指定服务类型
- 设定时间范围
## 平台（智能撮合）
- 匹配空闲技师
- 距离优先
- 评分筛选
## 店铺B（有余）
- 技师空闲登记
- 设置可借调时段
- 确认借调请求
## 服务完成后
### 店铺A
- 支付服务费
### 店铺B
- 获得分成
### 技师
- 工资+借调补贴
### 平台
- 收取撮合费
```

### 各方收益

| 角色 | 收益 |
|------|------|
| 借入店铺 | 解决用人荒，不流失订单 |
| 借出店铺 | 闲置资源变现，额外收入 |
| 技师 | 更多工作机会，收入增加 |
| 平台 | 撮合服务费，提升粘性 |

## 2. 评价可迁移

### 痛点解决

传统模式下，技师的评价：
- 绑定在店铺，换店清零
- 被店铺控制，可能被删除
- 不够全面，只有星级

### 我们的方案

**评价跟着技师走，不被店铺绑架**

| 特性 | 说明 |
|------|------|
| 永久保留 | 评价存储在平台，不随店铺变化 |
| 多维度 | 手法、态度、准时、沟通、卫生 |
| 不可删除 | 真实评价，店铺无法干预 |
| 可追溯 | 顾客可查看技师完整历史 |

### 评价维度


```mindmap
# 技师评分卡
## 专业技能 ★★★★★
- 手法熟练度
- 力度控制
- 穴位准确
## 服务态度 ★★★★☆
- 礼貌程度
- 沟通能力
- 耐心程度
## 准时守约 ★★★★★
- 是否准时到达
- 服务时长是否足够
## 沟通能力 ★★★★☆
- 需求理解
- 过程沟通
- 建议专业性
## 卫生习惯 ★★★★★
- 个人卫生
- 工具清洁
- 环境整理
```

## 3. T+N 结算机制

### 痛点解决

技师常见遭遇：
- 工资拖欠，讨薪困难
- 店铺跑路，血本无归
- 结算不透明，被克扣

### 我们的方案

**平台托管资金，保障技师权益**


```mindmap
# T+N 结算机制
## 订单完成
- 用户确认服务
- 系统记录完成
## 资金托管（核心）
- 进入平台托管账户
- 银行级安全保障
- 资金不可挪用
## T+N 自动结算
### 散技师 T+3
- 快速到账
- 降低入门门槛
- 吸引更多入驻
### 店铺 T+3
- 结算给店铺账户
- 店铺自行分配
- 技师可查看明细
## 动态周期调整
- 历史信用评估
- 服务评价记录
- 保证金额度
- 投诉纠纷情况
## 安全保障
- 银行托管账户
- 自动结算系统
- 争议冻结机制
```

### 结算规则详解

| 角色 | 结算周期 | 结算对象 | 说明 |
|------|---------|---------|------|
| 散技师 | T+3 | 技师本人 | 快速到账，降低入门门槛，吸引更多散技师入驻 |
| 店铺技师 | T+3 | 店铺账户 | 店铺自行管理分配，技师可查看平台订单明细 |

### 为什么是 T+3？

**对散技师**：
- 快速到账，增强吸引力
- 降低入门门槛，吸引更多优质技师入驻
- 留有足够时间应对顾客投诉等问题

**对店铺**：
- 资金直接结算给店铺，店铺有自己的结算周期和方案
- 店铺老板在平台约束范围内有分配灵活性
- 技师可查看所有平台关联订单的收入明细（作为凭证）

### 动态周期机制

T+N 的 N 值并非固定，而是根据以下因素动态调整：

| 评估因素 | 影响 |
|---------|------|
| 历史信用 | 信用良好可缩短周期 |
| 服务评价 | 高评分享受更快结算 |
| 保证金额度 | 缴纳保证金可加速 |
| 投诉记录 | 投诉多则延长周期 |

### 纠纷保障机制

当发生薪资纠纷时：

1. **资金冻结**：从店铺的 T+3 待结算资金中暂时冻结相关金额
2. **人工介入**：平台客服介入调查核实
3. **仲裁处理**：根据调查结果，决定是否将平台关联订单的提成部分调拨给技师
4. **凭证追溯**：技师可查看所有平台订单明细作为维权凭证

> 注：纠纷处理仅涉及平台关联订单，不包含店铺自有客源的项目收入

## 4. 散技师生态

### 什么是散技师？

不隶属于任何店铺，独立接单的技师。

### 平台支持

| 支持项 | 说明 |
|--------|------|
| 独立接单 | 直接接受用户订单 |
| 自主定价 | 自己设定服务价格 |
| 灵活时间 | 自己安排工作时间 |
| 评价积累 | 建立个人品牌 |
| 保险保障 | 平台提供意外险 |

### 与店铺技师对比

| 维度 | 店铺技师 | 散技师 |
|------|---------|--------|
| 收入 | 底薪+提成 | 纯提成，上限更高 |
| 自由度 | 受店铺约束 | 完全自主 |
| 稳定性 | 相对稳定 | 波动较大 |
| 适合人群 | 求稳定 | 求高收入、自由 |

## 6. 商户赋能服务

### 核心理念

**不是保证，而是赋能**

我们不承诺让店铺起死回生或一定赚钱，而是提供传统商家老板想要但因资金、技能、资源等原因难以实现的能力。

### 服务内容

#### 标准化服务

| 服务项 | 说明 |
|--------|------|
| 行业管理系统 | 按摩 SPA 行业深耕的业绩提成、结算统计、规划分析系统 |
| 顾客管理 | 会员档案、消费记录、余额管理、到期提醒 |
| 技师管理 | 排班、绩效、提成、考勤一体化 |
| 进销存管理 | 产品库存、消耗统计、采购提醒 |
| 数据报表 | 订单、收入、成本、利润多维度分析 |

#### 定制化服务

| 服务项 | 说明 |
|--------|------|
| 专业拍摄 | 店铺环境、技师形象、服务流程拍摄 |
| 平台运营 | 小红书、美团、抖音等平台代运营方案 |
| 品牌策划 | 店铺定位、视觉设计、营销策略 |
| 培训赋能 | 服务标准、销售技巧、管理能力培训 |

### 合作模式

**区别于传统服务商**

| 对比项 | 传统服务商 | 轻养到家 |
|--------|-----------|---------|
| 收费方式 | 先付费，后服务 | 评估后灵活定价 |
| 结果责任 | 不兜底，问题归咎商家 | 利益绑定，共担风险 |
| 服务深度 | 一次性交付 | 持续陪跑，深度参与 |

### 三种合作档位

```mindmap
# 商户赋能合作模式
## 保底+抽成模式
- 收取基础服务费
- 按新增订单抽成
- 适合大多数商户
## 纯抽成模式（零前置费用）
- 不收取任何前置费用
- 仅按订单业绩抽成
- 适合评估后的优质潜力店铺
- 平台深度绑定，利益共同体
## 战略入股模式
- 平台引入资金入股
- 深度参与经营决策
- 适合极具潜力的优质店铺
- 共同成长，共享收益
```

### 评估机制

平台会对商户进行综合评估，决定合作模式：

| 评估维度 | 考量因素 |
|---------|---------|
| 基础条件 | 地理位置、店铺规模、现有客源 |
| 经营状况 | 历史业绩、成本结构、盈利能力 |
| 团队能力 | 技师水平、服务质量、管理能力 |
| 发展潜力 | 市场空间、竞争环境、增长可能 |

**评估结论决定合作方式**：
- 可行性高、只缺通用性扶持 → 零前置费用，纯抽成
- 有潜力但需要投入 → 保底+抽成
- 极具潜力、值得深度绑定 → 战略入股

### 核心价值

**先干活，按业绩拿钱**

- 我们的收益来自为商家创造的增量价值
- 没有业绩增长，我们也没有收益
- 真正的利益共同体，而非甲乙方关系

### 经验积累

每一次深耕服务都是平台的经验积累：
- 沉淀行业最佳实践
- 优化服务方法论
- 形成可复制的赋能模式
- 反哺平台整体能力提升
